# esma-dev-studio.github.io

データビジュアライゼーション作品集のポータルページ。
**https://esma-dev-studio.github.io/**

実データで作るブラウザ3D作品(鉄道・太陽系・震源・星座・人工衛星)へのリンクをまとめた、
ビルド不要の静的1ページ(`index.html`)です。

## 更新方法

`index.html` を編集して main に push するだけ(GitHub Pages がユーザーページとして自動配信)。
新しい作品を追加するときは `.card` ブロックを1つ複製し、タイトル・説明・リンク・SVGサムネイルを差し替える。

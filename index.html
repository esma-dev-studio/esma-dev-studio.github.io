ESMA DEV STUDIO

DATA VISUALIZATIONS

実データで作るブラウザ3Dビジュアライゼーション作品集。 鉄道の地下深度から人工衛星の軌道まで、公開データを立体空間に描いています。 すべてPCブラウザでそのまま動きます(インストール不要)。

5 WORKS 100% OPEN DATA Three.js / React / TypeScript

東京・神奈川 鉄道路線 3DマップTOKYO × KANAGAWA RAIL 3D

48路線・約800駅を実際の線形で立体表示。地下鉄が地表の下に沈み、路線ごとの深さの違いや地下・高架の入れ替わりが見える。県境・海岸線つき。

国土数値情報 N02/N03地下×高架
アプリを開く GitHub

太陽系 立体マップSOLAR SYSTEM 3D

惑星・準惑星・主要衛星18天体を実際の軌道要素で配置。時間再生で公転・自転が動き、土星の環や天王星の横倒し自転も再現。天体を追跡できる。

NASA JPL 軌道要素時間再生
アプリを開く GitHub

日本の震源 3DマップJAPAN QUAKES 3D

25年分・16,620個の地震(M4.5+)を実際の深さで表示。太平洋プレートが日本の下へ沈み込む形が震源の並びとして浮かび上がる。時間再生では2011年3月が一斉に光る。

USGS 実深度データプレート境界
アプリを開く GitHub

星座の本当のかたちSTELLAR NEIGHBORHOOD 3D

約4万個の恒星を視差による実距離で立体配置。星座を選ぶと「地球から見た形」になり、回り込むと形が崩れていく——星座が方向の一致にすぎないことを体験できる。

HYG 恒星カタログ88星座
アプリを開く GitHub

人工衛星ライブ軌道マップSATELLITES 3D

稼働中の全衛星 約16,000機(スターリンク1万機超)をSGP4軌道計算でリアルタイム表示。慣性空間で地球が自転し、静止衛星の同期やISSの世界一周が見える。

CelesTrak TLEISS追跡
アプリを開く GitHub
